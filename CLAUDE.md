# KOMPASS HUB — Landing Page

## Produto
Kompass HUB — Plataforma B2B SaaS All-in-One brasileira para escritórios de advocacia e empresas B2B.

## Objetivo do Projeto
Landing page de alta conversão para gerar demos e assinaturas. Estética dark luxury-tech, digna de Awwwards.

## Stack
- HTML + CSS + Vanilla JS (arquivo único)
- Zero frameworks externos
- Google Fonts via @import: Clash Display + Instrument Sans

## Paleta de Cores

## Seções da Página
1. NAV — fixo, blur-backdrop
2. HERO — "O fim das abas infinitas." + ilustração SVG hub
3. TRUST BAR — marquee CSS com features
4. ECOSYSTEM TOUR — tabs interativas (CRM · Captação · Jurídico · Financeiro · Academy)
5. ONE-CLICK MOMENT — diagrama SVG animado WhatsApp → 4 módulos
6. DIFFERENTIALS — antes/depois + stats com count-up
7. SOCIAL PROOF — 3 depoimentos glassmorphism
8. FINAL CTA — "Pronto para operar no próximo nível?"
9. FOOTER

## Identidade Visual (versão final — navy/gold/white)
- Logo: duas triângulos (branco + dourado) formando agulha de bússola
- Paleta: navy (#0E1120) · gold (#B8A97A) · white. SEM teal/neon.
- Fontes: Cormorant Garamond (display) + DM Sans (corpo)
- Tom: luxo jurídico + tecnologia. Gravitas, precisão, elegância.
