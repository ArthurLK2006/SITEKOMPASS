# KOMPASS HUB — Landing Page

## Produto
Kompass HUB — Plataforma B2B SaaS All-in-One brasileira para escritórios de advocacia e empresas B2B.

## Objetivo do Projeto
Landing page de alta conversão para gerar demos e assinaturas. Estética dark luxury-tech, digna de Awwwards.

## Stack
- HTML + CSS + Vanilla JS (arquivo único)
- Zero frameworks externos
- Google Fonts via @import: Clash Display + Instrument Sans
